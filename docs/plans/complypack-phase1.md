# ComplyPack Phase 1: OPA Native Provider

Implementation plan for the OPA evaluation provider that consumes ComplyPack content.

**Status:** not-started

**Related ADRs:** [0004](../ADRs/0004-grpc-provider-plugin-architecture.md), [0006](../ADRs/0006-complypack-content-envelope.md)

**Depends on:** [Phase 0](complypack-phase0.md)

## Goal

Ship a native OPA provider that receives ComplyPack content from the runtime client, evaluates it against collected data, and returns structured assessment results.
