# Evidence

Compliance evidence is fragmented, manual, and opaque. Every assessment produces different artifacts. There is no standard way to collect, normalize, or trace evidence back to the requirement it satisfies.

This problem area is under active exploration. See [evaluator-coupling.md](evaluator-coupling.md) for how collection and evaluation conflation affects evidence, and [requirement-fidelity.md](requirement-fidelity.md) for how fidelity loss weakens the link between evidence and requirements.
